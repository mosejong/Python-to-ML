## 오늘 배운 내용

### 1. TensorFlow와 MLP
- TensorFlow를 활용해서 다층 퍼셉트론(MLP) 모델을 구성했다.
- 입력층, 은닉층, 출력층의 구조를 이해했다.
- 출력층에서 다중 분류 문제일 경우 `softmax`를 사용한다는 점을 다시 확인했다.
- `model.add(layers.Dense(num_classes, activation='softmax'))` 코드는 최종 출력층을 의미하며, 클래스별 확률을 출력하는 역할을 한다.

### 2. 모델 결과 비교와 앙상블 감각 익히기
- 서로 다른 모델 결과를 평균내는 방식이 있다는 것을 배웠다.
- 각각의 예측 결과를 따로 비교하는 것뿐 아니라, 여러 결과를 섞어서 더 안정적인 점수를 만드는 방식도 가능하다는 걸 이해했다.
- 제출 점수를 보면서 어떤 방식이 더 좋은 결과를 내는지 비교해봤다.

### 3. 제출 결과 확인
- 여러 번 제출하면서 점수를 비교했다.
- 단순히 모델을 만드는 것에서 끝나는 게 아니라, 제출 결과를 보고 개선 방향을 잡는 과정도 중요하다는 걸 느꼈다.

## 기억할 점
- 딥러닝 모델은 구조를 만드는 것도 중요하지만, 결과를 비교하고 개선하는 과정이 더 중요하다.
- `softmax`는 다중 분류에서 각 클래스 확률을 출력한다.
- 예측값 평균 같은 앙상블 방식은 성능 개선에 도움이 될 수 있다.

## 회고
- 오늘은 TensorFlow와 MLP 흐름을 다시 정리할 수 있었다.
- 제출 결과 숫자를 직접 보면서 실험하는 과정이 재미있었다.
- 단순 코드 작성보다 "왜 이렇게 결과가 나오는지" 생각하는 습관이 더 중요하다고 느꼈다.
