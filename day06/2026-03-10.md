## 1. 정규표현식 (Regular Expression)

문자열에서 특정 패턴을 찾거나 치환하기 위해 사용하는 표현식이다.  
로그 분석, 데이터 전처리, 문자열 검색 등에 많이 사용된다.

사용 모듈

```python
import re
```

---

## 2. 메타문자

정규표현식에서 특별한 의미를 가지는 문자

```
. ^ $ * + ? { } [ ] \ | ( )
```

| 메타문자 | 의미 |
|------|------|
| . | 임의의 문자 |
| ^ | 문자열 시작 |
| $ | 문자열 끝 |
| * | 0번 이상 반복 |
| + | 1번 이상 반복 |
| ? | 0 또는 1번 |
| { } | 반복 횟수 지정 |
| [ ] | 문자 집합 |
| \ | 이스케이프 문자 |
| \| | OR 연산 |
| ( ) | 그룹 |

---

# 🔍 정규식을 이용한 문자열 검색

### match()

문자열 **처음부터 정규식과 매치되는지 검사**

```python
re.match()
```

---

### search()

문자열 **전체를 검색하여 정규식과 매치되는지 검사**

```python
re.search()
```

---

### findall()

정규식과 매치되는 모든 문자열을 **리스트로 반환**

```python
re.findall()
```

---

### finditer()

정규식과 매치되는 모든 문자열을 **반복 가능한 객체(iterator)** 로 반환

```python
re.finditer()
```

# 📌 전방 탐색 (Lookahead)

뒤에 오는 패턴을 **소비하지 않고 검사**

### 긍정 전방 탐색


### 부정 전방 탐색

# 🧪 코드 실습

## DashInsert 문제

홀수 뒤에 홀수가 오면 `-` 삽입  
짝수 뒤에 짝수가 오면 `*` 삽입

```python
import re

def DashInsert(a):
    a = re.sub(r'([13579])(?=[13579])', r'\1-', a)
    a = re.sub(r'([2468])(?=[2468])', r'\1*', a)
    return a

print(DashInsert('4546793'))
```

# 🧪 문자열 압축 문제

문자열

```
aaabbccccca
```

결과

```
a3b2c5a1
```

```python
import re

a = 'aaabbccccca'
result = ''

for m in re.finditer(r'(.)\1*', a):
    result += m.group()[:1] + str(len(m.group()))

print(result)
```

정규식 설명

```
(.)  → 아무 문자 하나
\1*  → 같은 문자 반복
```

매칭 결과

```
aaa
bb
ccccc
a
```
