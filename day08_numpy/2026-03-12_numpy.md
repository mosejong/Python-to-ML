## NumPy
NumPy는 Python에서 다차원 배열을 빠르게 계산하기 위한 라이브러리이며 데이터 분석, AI, 머신러닝에서 많이 사용된다.

import numpy as np

## 배열 생성
arr = np.arange(10)

## 배열 형태 변경
arr = np.arange(12).reshape(3,4)

## 인덱싱 / 슬라이싱
arr[1]       # 특정 행
arr[:,1]     # 특정 열
arr[1:3,1:4] # 부분 배열

## Boolean Indexing
arr = np.arange(10)
arr[arr % 2 == 0]

## 브로드캐스팅
arr = np.arange(5)
arr_1 = np.arange(5).reshape(5,1)
arr + arr_1

## 집계 함수
arr = np.arange(1,13).reshape(3,4)
arr.sum()
arr.mean()
arr.sum(axis=0)
arr.sum(axis=1)

## 배열 정렬
np.sort(arr, axis=1)

## 배열 결합
np.concatenate([A,B], axis=0)
np.vstack((A,B))
np.hstack((A,B))

## 행렬 연산
np.dot(A,B)
A @ B

## 역행렬
np.linalg.inv(A)

오늘 NumPy에서 배열 생성, 인덱싱, 브로드캐스팅, 집계, 배열 결합, 행렬 연산을 학습하였다.
NumPy는 벡터 연산을 통해 반복문 없이 빠른 계산을 수행할 수 있다.
