### 2. 반복문을 활용한 간단한 프로그램
반복문을 사용하여 간단한 프로그램을 만들 수 있다.

예: 피보나치 수열

```python
A = [0,1]
n = int(input('피보나치함수 - 정수입력 : '))
i = 0

while A[i] + A[i+1] <= n:
    A.append(A[i] + A[i+1])
    i += 1

print(A)
```

### 4. 하위 디렉터리 검색

```python
import os

for (path, dir, files) in os.walk("C:\msj"):
    print(path)
    print(dir)
    print(files)

    for filename in files:
        ext = os.path.splitext(filename)[-1]

        if ext == '.py':
            print('%s==>%s' % (path,filename))
```

### 5. 문자 인코딩
Python 문자열은 **UTF-8 기반 유니코드**를 사용한다.

```
encode = 문자열 → 바이트
decode = 바이트 → 문자열
```

```python
text = "hello"
encoded = text.encode("utf-8")
decoded = encoded.decode("utf-8")
```

### 7. 데코레이터 (Decorator)

### 8. 이터레이터 (Iterator)
`next()`를 통해 값을 하나씩 꺼낼 수 있는 객체

```python
a = iter([1,2,3])
next(a)
```

### 9. 제너레이터 (Generator)
`yield`를 사용하여 값을 하나씩 생성하는 함수

```python
def gen():
    yield 1
    yield 2
```
