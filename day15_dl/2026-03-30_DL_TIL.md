# DL | 딥러닝 개론

## 📝 오늘 배운 내용 요약

1. 딥러닝 개론
    - 딥러닝은 머신러닝의 여러 방법론 중 하나이다.
    - 인공신경망(Artificial Neural Network)을 기반으로 컴퓨터가 사람의 사고방식을 학습하도록 만드는 방법이다.
    - 인공신경망은 생물학적 신경망에서 영감을 받아 만들어졌으며, 사람의 뉴런 시스템을 모방한 학습 알고리즘이다.
    - 모델이 스스로 데이터의 특징을 학습할 수 있어 지도학습, 비지도학습 모두에 활용될 수 있다.

2. 퍼셉트론(Perceptron)
    - 퍼셉트론은 신경망의 기본이 되는 개념이다.
    - 기존의 명시적 프로그래밍은 사람이 규칙을 일일이 입력해야 한다는 한계가 있다.
    - 자율주행 자동차처럼 복잡한 문제는 사람이 모든 상황을 직접 규칙으로 정의하기 어렵기 때문에, 기계가 스스로 학습하는 방식이 필요하다.
    - 퍼셉트론은 입력값과 가중치를 곱한 뒤 모두 더하고, 편향(Bias)을 더한 값을 활성화 함수에 통과시켜 결과를 출력한다.
    - 수식으로는 `y = activation(w1x1 + w2x2 + b)` 형태로 표현할 수 있다.

3. 퍼셉트론과 선형 분류
    - 퍼셉트론은 선형 분류기로 사용할 수 있다.
    - 하나의 직선으로 데이터를 나눌 수 있는 문제를 분류할 수 있다.
    - AND, OR, NAND, NOR 같은 논리 게이트는 퍼셉트론으로 표현 가능하다.
    - XOR gate처럼 하나의 직선으로 나눌 수 없는 비선형 문제는 단층 퍼셉트론으로 해결하기 어렵다.

4. 다층 퍼셉트론(MLP)
    - 다층 퍼셉트론은 단층 퍼셉트론을 여러 개 쌓은 구조이다.
    - 입력층과 출력층 사이에 존재하는 층을 히든층(Hidden Layer)이라고 한다.
    - 히든층이 늘어나면 더 복잡한 문제를 학습할 수 있다.
    - 히든층이 깊어질수록 딥러닝이라는 표현을 사용한다.

5. 신경망의 구성요소
    - 입력층(Input Layer): 데이터를 처음 받는 층
    - 히든층(Hidden Layer): 입력 데이터를 가공하고 특징을 학습하는 층
    - 출력층(Output Layer): 최종 결과를 출력하는 층
    - 가중치(Weight): 각 입력값이 결과에 미치는 중요도
    - 편향(Bias): 결과를 조정하는 값
    - 활성화 함수(Activation Function): 입력값을 변환하여 다음 층으로 전달하는 함수

6. 텐서(Tensor)
    - 텐서는 딥러닝에서 데이터를 표현하는 기본 단위이다.
    - 스칼라(0차원), 벡터(1차원), 행렬(2차원)을 포함하는 더 일반적인 다차원 배열 개념이다.
    - 딥러닝에서는 이미지, 텍스트, 수치 데이터 등을 텐서 형태로 변환하여 처리한다.
    - TensorFlow의 Tensor는 단순한 배열이 아니라 딥러닝 연산에 적합한 자료구조이다.
    - numpy 배열과 비슷하지만, 텐서는 GPU 연산과 자동미분 같은 딥러닝 기능에 더 적합하다.

7. 실습과 연결한 이해
    - 머신러닝 모델(RandomForest, MLPClassifier, Perceptron)과 Keras 모델을 비교하며 분류 문제를 실습했다.
    - 머신러닝 코드는 이전보다 익숙해져서 안 보고도 직접 입력할 수 있었다.
    - Keras 코드는 아직 완전히 이해되지는 않지만, 전체 흐름이 `전처리 → 라벨 인코딩 → 모델 생성 → compile → fit → evaluate` 순서라는 점을 복습했다.
    - 학습 정확도와 테스트 정확도가 다를 수 있으며, 학습 정확도만 높고 테스트 정확도가 낮다면 과적합을 의심할 수 있다는 점을 확인했다.
    - 이미지 데이터라고 해서 항상 딥러닝이 더 좋은 것은 아니며, 데이터 양이 적을 경우 RandomForest 같은 전통 머신러닝 모델이 더 좋은 성능을 낼 수 있다는 점도 확인했다.

## 💭 오늘의 회고

1. **배운 점**
    - 딥러닝은 인공신경망을 기반으로 하는 학습 방법이라는 점을 이해했다.
    - 퍼셉트론이 선형 분류의 기본 개념이라는 점을 배웠다.
    - XOR 같은 비선형 문제를 해결하기 위해 다층 퍼셉트론이 필요하다는 흐름을 이해했다.
    - 텐서는 딥러닝에서 사용하는 기본 데이터 단위이며, numpy 배열과 비슷하지만 딥러닝 계산에 더 적합한 자료구조라는 점을 알게 되었다.
    - 이미지 분류 문제에서도 데이터 양과 구조에 따라 전통 머신러닝 모델이 더 강할 수 있다는 점을 확인했다.

2. **어려운 점/개선할 점**
    - Keras 코드는 아직 복붙한 부분이 많아 완전히 내 것으로 느껴지지는 않았다.
    - `compile`, `fit`, `evaluate`의 의미를 더 자연스럽게 설명할 수 있도록 복습이 필요하다.
    - 텐서와 numpy 배열의 차이, 그리고 왜 DataFrame을 numpy로 바꿔서 입력하는지 더 익숙해질 필요가 있다.
    - train accuracy와 validation/test accuracy의 차이를 보며 과적합을 판단하는 연습이 더 필요하다.
